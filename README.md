Welcome to the OpsRamp OpenAPI repository on GitHub.

## Requirements

- A local text editor or IDE, such as **[VS Code](https://code.visualstudio.com/)** or **[Sublime Text](https://www.sublimetext.com/)**
- **[Git](https://git-scm.com/book/en/v2/Getting-Started-Installing-Git)**, or a Git Client such as [Tower](https://www.git-tower.com/mac)
- **[Hugo](https://gohugo.io/getting-started/installing/)**
- **[Node.js](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm)**
- Optional: [Yarn](https://classic.yarnpkg.com/en/docs/install/#mac-stable)
